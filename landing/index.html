Everything you need to build better RAG

Stop guessing. Start measuring. Comprehensive tools for every stage of your pipeline.

Hallucination Detection

Automatically detect when your LLM makes up facts not present in the retrieved context using our specialized faithfulness metrics.

CI/CD Integration

Block bad deployments before they reach production. RAGLint integrates seamlessly with GitHub Actions to catch regressions.

Cost & Latency Tracking

Monitor token usage and response times across different models and providers. Optimize for both quality and budget.

Visual Dashboard

Explore test runs, compare model versions, and debug individual queries with our beautiful, built-in web dashboard.

Auto-Instrumentation

Just add the @raglint.watch decorator. No complex setup required to start capturing traces.

LLM Integrations

First-class support for OpenAI, Azure, Bedrock, and LangChain out of the box.

A/B Testing

Compare different pipeline versions side-by-side. Track performance improvements over time.

Self-Hosted

Deploy on your own infrastructure with Docker. Keep full control over your sensitive data.

Drop-in Integration

Add RAGLint to your existing Python pipeline in less than 5 minutes. No architecture changes required.

1

Install the package

Available on PyPI for Python 3.9+. Just pip install and you're ready.

2

Wrap your chain

Use our LangChain integration or manual hooks to instrument your code.

3

View the dashboard

Launch the UI to see your insights instantly and debug issues.

main.py
from raglint import RAGLint
from raglint.integrations import LangChainEvaluator

# 1. Initialize RAGLint
evaluator = LangChainEvaluator(my_qa_chain)

# 2. Run evaluation
results = await evaluator.evaluate(test_questions)

# 3. Launch dashboard
# $ raglint dashboard
