How it Works

RAGLint is designed to be drop-in compatible with your existing Python RAG pipelines.

  • 1

    Install the package

    Available on PyPI for Python 3.9+

  • 2

    Wrap your chain

    Use our LangChain integration or manual hooks.

  • 3

    View the dashboard

    Launch the UI to see your insights instantly.

main.py
from raglint import RAGLint
from raglint.integrations import LangChainEvaluator

# 1. Initialize RAGLint
evaluator = LangChainEvaluator(my_qa_chain)

# 2. Run evaluation
results = await evaluator.evaluate(test_questions)

# 3. Launch dashboard
# $ raglint dashboard
