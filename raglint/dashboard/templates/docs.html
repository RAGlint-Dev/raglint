What is RAGLint?

RAGLint is a comprehensive evaluation framework for Retrieval-Augmented Generation (RAG) systems. It provides production-ready tools to measure, monitor, and improve your RAG pipelines.

Smart Metrics

Faithfulness, semantic similarity, context precision/recall, and more.

Plugin System

Extensible architecture for custom metrics and evaluators.

Dashboard

Beautiful web UI for visualizing results and debugging queries.

Cost Tracking

Monitor LLM costs and latency across different providers.

CLI & Python API

Use from command line or integrate directly into your code.

Benchmarks

Test against SQUAD, CoQA, and other standard datasets.

ðŸŽ¯ Basic Usage

Python
from raglint import RAGPipelineAnalyzer

# Initialize analyzer with smart metrics
analyzer = RAGPipelineAnalyzer(use_smart_metrics=True)

# Prepare your test data
data = [
    {
        "query": "What is retrieval-augmented generation?",
        "retrieved_contexts": [
            "RAG combines retrieval and generation..."
        ],
        "response": "RAG is a technique that enhances LLMs..."
    }
]

# Run analysis
results = analyzer.analyze(data)

# View metrics
for result in results:
    print(result["metrics"])

LangChain

Seamless integration with LangChain chains and agents.

from raglint.integrations import LangChainEvaluator

evaluator = LangChainEvaluator(qa_chain)
results = evaluator.evaluate(questions)

LlamaIndex

Track and evaluate LlamaIndex query engines.

from raglint.integrations import LlamaIndexHandler

handler = LlamaIndexHandler()
index.as_query_engine(callbacks=[handler])

Haystack

Monitor Haystack pipelines with RAGLint.

from raglint.integrations import HaystackTracer

pipeline.run(query, callbacks=[HaystackTracer()])
