What is RAGLint?

RAGLint is a comprehensive evaluation framework for Retrieval-Augmented Generation (RAG) systems. It bridges the gap between prototype and production by providing robust tools to measure faithfulness, context relevance, and answer quality.

Why RAGLint?

Most RAG evaluations are either too simple (exact match) or too expensive (human review). RAGLint uses "Smart Metrics" – LLM-as-a-Judge – to provide human-level evaluation at machine speed.

Installation

RAGLint is available on PyPI and can be installed via pip.

bash
pip install raglint

Quick Start

Run your first analysis in just a few lines of code.

Python
from raglint import RAGPipelineAnalyzer

# 1. Initialize
analyzer = RAGPipelineAnalyzer(use_smart_metrics=True)

# 2. Analyze
results = analyzer.analyze([{
    "query": "What is RAG?",
    "response": "RAG is...",
    "retrieved_contexts": ["..."]
}])

# 3. View Results
print(results)
