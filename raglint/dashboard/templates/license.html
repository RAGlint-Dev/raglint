Frequently Asked Questions

Can I use RAGLint in my startup?

Yes, as long as your startup's application is open-source under AGPL-3.0. If you want to keep your code proprietary, you'll need a commercial license.

What if I modify RAGLint?

Any modifications must also be released under AGPL-3.0 if you distribute the software or run it as a network service accessible to users.

Can I use RAGLint internally?

Yes, internal use is allowed. However, if you provide RAGLint as a service to external users (even indirectly), AGPL-3.0 applies.
